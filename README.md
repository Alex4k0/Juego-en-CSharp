# Juego en C#
Echoes of the Void – Mini juego de combate en consola (C#)

Proyecto académico desarrollado en C# que simula un combate por turnos entre un héroe y varios enemigos. El objetivo es practicar programación orientada a objetos, herencia y lógica de juego en consola.

Características principales:

Sistema de clases basado en herencia: Entidad, Heroe, Enemigo, Orco, Hechicero.

Combate por turnos entre el jugador y los enemigos.

Acciones personalizadas según el tipo de entidad: el héroe realiza ataques críticos, el orco ataca directamente, el hechicero puede atacar o curarse.

Sistema de experiencia: el héroe gana EXP al derrotar enemigos.

Interacción mediante consola.

Tecnologías utilizadas:
C#
Programación orientada a objetos
Aplicación de consola (.NET)

Estructura del proyecto:
Entidad.cs
Enemigo.cs
Heroe.cs
Orco.cs
Hechicero.cs
Program.cs

Cómo ejecutar el proyecto:

Clonar el repositorio.

Abrir el proyecto en Visual Studio o VS Code.

Ejecutar.

Objetivo del proyecto:
Practicar clases y objetos, herencia, métodos virtuales, encapsulamiento, lógica de combate e interacción básica con el usuario.

Mejoras futuras:
